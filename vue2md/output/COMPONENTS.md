##  Form

### Input.vue

This component is custom input to be used with forms.

#### Parameters

*   `name` **[string][4]**&#x20;
*   `value` **[string][4]**  (optional, default `""`)
*   `type` **[string][4]**  (optional, default `"text"`)

#### Examples

```javascript
{
name : "my subtitle"
value : "P@ssw0rd"
type : "password"
}
```
